# Smart-Collect - AI Service 🚛🧠

Micro-service Python basé sur **FastAPI** qui expose les algorithmes intelligents pour la plateforme de gestion de collecte d'ordures de **Hysacam**.

## 🚀 Stack Technique
- **Python 3.11**
- **FastAPI 0.110+**
- **Docker & Docker-Compose**
- **Pytest** (Tests unitaires et asynchrones)
- **Structlog** (Journalisation JSON structurée)
